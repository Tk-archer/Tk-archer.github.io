Material Design Lite学习其一

发表于 2015-10-29

Material Design Lite组件的用法
官网连接在这里

badges

类名 作用 备注
mdl-badge 定义一个 MDL badge 容器 必须是span 或者link元素
mdl-badge–no-background 取消badge的背景颜色 可选
mdl-badge–overlap Make the badge overlap with its container 可选
data-badge=”value” badge 的值 不是一个类，而是一个属性值

button

类名 作用 备注
mdl-button 定义一个 mdl 按钮 必须的
mdl-js-button 指定基础的mdl按钮行为 必须的
mdl-button–raised 指定为矩形样式 与fab min-fab 和icon互斥
mdl-button–fab 指定为fab（圆形）样式 与raised, mini-fab和icon互斥
mdl-button–mini-fab 指定为min-fab（比上面小的圆形）样式 与 raised, fab和icon互斥
mdl-button–icon 指定为图标（小而平扁的）样式 与raised, mini-fab和fab互斥
mdl-button–colored 指定按钮是彩色 颜色定义在material.min.css
mdl-button–primary 指定按钮为基本色 颜色定义在material.min.css
mdl-button–accent 指定按钮为强调色 颜色定义在material.min.css
mdl-js-ripple-effect 显示波纹点击效果 可以结合其他任何类使用的

card

类名 作用 备注
mdl-card 定义一个MDL card容器 必须是最外层的div
mdl-card–border 用于添加card容器的边框 用在内部的div中
mdl-shadow–2dp through mdl-shadow–16dp 指定card容器的阴影深度（2,3,4,6,8,16） 可选的，用于外层div 如果省略，则没有阴影
mdl-card__title 定义div为card的标题容器 必须是内部的标题div
mdl-card__title-text 指定对应的card标题文字 必须是用于 标题div内部head标签（h1~h6）
mdl-card__subtitle-text 指定对应的card副标题文字 可选的，应该是是标题元素的子元素
mdl-card__media 定义一个card的媒体容器 必须用于内部的媒体div
mdl-card__supporting-text 定义一个card文本容器，并分配相应的文本特性与正文 必须是用于内部的正文div，正文直接插入这个元素中，不需要其他内部容器
mdl-card__actions 定义一个card的交互容器并分配相应的文本特征给动作的文本 必须是用于内部的交互div，交互内容直接插入这个元素中，不需要其他内部容器

loading

Progress bar

类名 作用 备注
mdl-progress 定义一个MDL 进度条组件容器 必须的
mdl-js-progress 指定一个基础的MDL 进度指示器交互行为 必须的
mdl-progress__indeterminate 增加动态效果 可选的

spinner

类名 作用 备注
mdl-spinner 定义一个MDL spinner组件容器 必须的
mdl-js-spinner 指定一个基础的MDL spinner交互行为 必须的
is-active 令spinner可见并执行动画效果 可选的
mdl-spinner–single-color 使用单一颜色而不改变颜色 可选的

menu

类名 作用 备注
mdl-button 定义一个 mdl 按钮 必须的
mdl-js-button 指定基础的mdl按钮行为 必须的
mdl-button–icon 指定为图标（小而平扁的）样式 必须的
material-icons 定义一个MDL icon的span元素 必须是一个行内元素
mdl-menu 定义一个MDL无序列表组件 必须是ul元素
mdl-js-menu 指定基础的MDL 菜单交互行为 必须是ul元素
mdl-menu__item 定义button为菜单选项以及指定基础的MDL交互样式 必须是 list item元素
mdl-js-ripple-effect 为菜单选项加上波纹点击效果 可选，无序列表元素
mdl-menu–top-left 定位菜单在按钮之上, 与按钮左边缘对齐 可选，无序列表元素
(none) 定位菜单在按钮的下边, 与按钮左边缘对齐 默认
mdl-menu–top-right 定位菜单在按钮的上边, 与按钮右边缘对齐 可选，无序列表元素
mdl-menu–bottom-right 定位菜单在按钮的下边, 与按钮右边缘对齐 可选，无序列表元素

slider

类名 作用 备注
mdl-slider 定义一个input 元素为MDL silder组件 必须的
mdl-js-slider 指定MDL silder组件的基础交互样式 必须的

toggles

Checkbox

类名 作用 备注
mdl-checkbox 定义label为一个MDL checkbox组件 必须是一个label元素
mdl-js-checkbox 指定MDL checkbox组件的基础交互行为 必须是一个label元素
mdl-checkbox__input 指定MDL checkbox输入框的基础交互行为 必须是一个label元素 (checkbox)
mdl-checkbox__label 指定MDL checkbox组件说明标签的基础交互行为 必须是一个span元素 (caption)
mdl-js-ripple-effect 添加波纹点击效果 可选用在label元素而不是input元素(checkbox)上

radio button

类名 作用 备注
mdl-radio 定义label为一个MDL radio组件 必须是一个label元素
mdl-js-radio 指定MDL radio组件的基础交互行为 必须是一个label元素
mdl-radio__button 为radio元素添加MDL 交互行为 必须是一个input元素 (radio button)
mdl-radio__label 指定MDL radio组件说明标签的基础交互行为 必须是一个span元素 (caption)
mdl-js-ripple-effect 添加波纹点击效果 可选用在label元素而不是input元素(checkbox)上

icon toggle

类名 作用 备注
mdl-icon-toggle Defines label as an MDL component Required on label element
mdl-js-icon-toggle Assigns basic MDL behavior to label Required on label element
mdl-icon-toggle__input Applies basic MDL behavior to icon-toggle Required on input element (icon-toggle)
mdl-icon-toggle__label Applies basic MDL behavior to caption Required on i element (icon)
mdl-js-ripple-effect Applies ripple click effect Optional; goes on label element, not input element (icon-toggle)

switch

类名 作用 备注
mdl-switch Defines label as an MDL component Required on label element
mdl-js-switch Assigns basic MDL behavior to label Required on label element
mdl-switch__input Applies basic MDL behavior to switch Required on input element (switch)
mdl-switch__label Applies basic MDL behavior to caption Required on span element (caption)
mdl-js-ripple-effect Applies ripple click effect Optional; goes on label element, not input element (switch)

table

类名 作用 备注
mdl-data-table 定义一个MDL table组件 需要用于table元素
mdl-js-data-table 指定基础MDL table 交互行为 需要用于table元素
mdl-data-table–selectable 使表格元素可选 可选 用于table元素
mdl-data-table__cell–non-numeric 格式化数据单元格 可选 用于表头和表单数据单元格

text fields

类名 作用 备注
mdl-textfield 定义一个MDL组件容器 必须是用于外层div元素
mdl-js-textfield 指定基础的MDL input元素行为 必须是用于外层div元素
mdl-textfield__input 定义一个文本域输入元素 必须是用于input 或者textarea元素
mdl-textfield__label 定义一个文本域标签元素 必须是用于绑定了input 或者textarea的label元素
mdl-textfield–floating-label 添加label浮动效果 可选，在外层的div元素
mdl-textfield__error 定义一个MDL错误信息的span 用在有pattern属性的MDL input元素之后的span元素
mdl-textfield–expandable 定义一个MDL可扩展的文本域容器 为了可扩展输入域，用于外层容器
mdl-button 按钮 为了可扩展输入域，用于外层容器下的label元素
mdl-js-button 按钮交互 为了可扩展输入域，用于外层容器下的label元素
mdl-button–icon 图标按钮 为了可扩展输入域，用于外层容器下的label元素
mdl-input__expandable-holder 定义一个MDL组件容器 为了可扩展输入域，用于内部的div元素

tooltip

用例

类名 作用 备注
mdl-tooltip 定义一个 MDL tooltip容器 用于 tooltip容器元素
mdl-tooltip–large 更大的文字效果 可选 用于 tooltip容器元素

layout

grid

类名 作用 备注
mdl-grid 定义一个MDL grid组件的容器 用于外层的div元素
mdl-cell 定义一个MDL cell容器 必须用于内部的div容器
mdl-grid–no-spacing 清除cell之间的margin值 可选，用于grid容器
mdl-cell–N-col 设置cell所占的列大小为N N的值包括1-12，默认是4 用于内部div元素
mdl-cell–N-col-desktop 单独设置desktop模式cell所占的列大小为N N的值包括1-12，用于内部div元素
mdl-cell–N-col-tablet 单独设置tablet模式cell所占的列大小为N N的值包括1-8，用于内部div元素
mdl-cell–N-col-phone S单独设置phone模式cell所占的列大小为N N的值包括1-4，用于内部div元素
mdl-cell–hide-desktop 在desktop模式下隐藏该cell 可选，用于内部div元素
mdl-cell–hide-tablet 在tablet模式下隐藏该cell 可选，用于内部div元素
mdl-cell–hide-phone 在phone模式下隐藏该cell 可选，用于内部div元素
mdl-cell–stretch 垂直拉伸cell填满父元素 默认值; 用于内部div元素
mdl-cell–top 令cell对齐 父元素顶部 可选 用于内部div元素
mdl-cell–middle 令cell对齐 父元素中部 可选 用于内部div元素
mdl-cell–bottom 令cell对齐 父元素底部 可选 用于内部div元素
