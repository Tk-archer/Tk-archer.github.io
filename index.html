TCP协议

定义

TCP（Transmission Control Protocol 传输控制协议）是一种面向连接的、可靠的、基于字节流的传输层通信协议，由IETF的RFC 793定义。

在简化的计算机网络OSI模型中，它完成第四层传输层所指定的功能，用户数据报协议（UDP）是同一层内[1] 另一个重要的传输协议。

网络分层

总体划分

ISO提出的OSI（Open System Interconnection）模型将网络分为七层，即

  • 物理层( Physical )
  • 数据链路层(Data Link)
  • 网络层(Network)
  • 传输层(Transport)
  • 会话层(Session)
  • 表示层(Presentation)
  • 应用层(Application)。
